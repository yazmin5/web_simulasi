Pemodelan Pergerakan Manusia Pada Kereta Api Terhadap COVID-19 Menggunakan Algoritme Distance Decay PageRank

Distance-Decay PageRank dikembangkan oleh Chin dan Wen (2015) dengan menambahkan parameter jarak geografis pada algoritme PageRank. Parameter jarak akan menentukan bobot yang digunakan sebagai pertimbangan pada perhitungan skor PageRank. Efek distance-decay berasal dari hukum pertama geografi Tobler, “Semua hal itu terhubung, tetapi semakin dekat jaraknya, semakin dekat juga hubungannya” (Yin et al. 2019). Dalam jaringan geospasial, Chin dan wen (2015) menggunakan jarak geografis antar node sebagai sebuah bobot perjalanan yang dilakukan oleh manusia. Bobot ini dapat dijadikan faktor eksternal maupun internal dari sebuah node. Untuk menangkap efek Distance-Decay, Chin dan wen (2015) menggunakan nilai invers dari jarak node yang dituju ke node awal sebagai parameter pertimbangan pada perhitungan skor PR, kemudian skor akan dikirimkan dari node awal ke node yang dituju.

Formula dari Distance-Decay PageRank dapat dilihat pada Persamaan (1) dan (2) (Chin dan Wen 2020). Untuk setiap node a, yang mempunyai link dari node b (b∈Ia) dan untuk setiap node b, yang mempunyai link yang menuju ke node c (c∈Ob, dan a∈Ob).

DDPRt(a) adalah nilai skor PR dari node a pada iterasi t. Nilai skor inilah yang nantinya akan digunakan pada saat pemberian peringkat pada node. β merupakan Distance factor dan distance adalah jarak antara node i dan node j. Faktor jarak pada persamaan (2) merepresentasikan skala dari efek distance-decay. Semakin besar faktor jarak, efek distance-decay akan lebih signifikan. Pada saat faktor jarak bernilai 0, hasil dari perhitungan akan sama dengan algoritme awal, algoritme PageRank.

Pada pemodelan ini, Algoritme Distance-decay Pagerank digunakan untuk memodelkan pergerakan manusia pada transportasi kereta api di Provinsi Jawa Timur.
